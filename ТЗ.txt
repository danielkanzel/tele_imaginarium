#create

Создается объект игры
Создателю возвращается идентификатор игры
Игра в статусе starting

#join

К объекту игры добавляются игроки по идентификатору
Ведущему отображается информация о присоединившихся

#begin

Игра переводится в статус in_game
Новые игроки не могут присоединиться
Игрокам распределяются карточки
Создается список stats в объекте game:

[
    {
        "player":"1231234",     #id пользователя
        "pos":0,                #порядок хода в игре
        "cards":[1,2,3,4,5,6],  #набор карт
        "points":0              #кол-во очков
    },{
        "player":"1423459",
        "pos":1,
        "cards":[7,8,9,10,11,12],
        "points":0
    }
]

Всем игрокам рассылается набор карточек и указание следующего ходящего

#choose {номер}

Ведущему приходит запрос на ввод номера карточки

#guess {текст}

Ведущему приходит запрос на ввод текста
Ведущий вводит текст
Всем рассылается текст ведущего

#suggest {номер}

Игроки отправляют номера своих карточек
Нельзя отправить больше одной карточки
В базу собирается набор карточек на ходе
Формат хранения ходов:

[
    {
        "turn": 1,
        "phrase": "Кто в теремочке живет?",
        "cards": [
            {
                "player":"12132123",
                "real":True,
                "card":12
            },{
                "player":"42435233",
                "real":False,
                "card":8
            },{
                "player":"83573573",
                "real":False,
                "card":5
            }
        ]
    }
]

Когда количество загаданных карточек 
становится равно количеству игроков,
начинается подсчет очков.

Отображается карточка загаданная ведущим
Отображается рассчитанное количество очков
Инкрементируется номер ведущего
начинается новый ход
